# 세계사 시뮬레이터

브라우저에서 바로 실행 가능한 싱글페이지 전략 시뮬레이터입니다.

## 현재 포함 기능
- 3개 플레이 가능 세력과 세력별 고유 교리
- 12턴 캠페인 구조
- 정책 카드 선택
- 대외 행동 선택: 연구 협정, 자원 교환, 동맹, 제재, 압박, 제한전
- 4개 국가 스탯: 안정성, 경제력, 군사력, 기술력
- 3개 핵심 자원: 에너지, 연산 자원, 정당성
- AI 세력 자동 행동 시뮬레이션
- 세계 긴장도 시스템
- 관계 상태 시스템: 중립, 우호, 동맹, 전쟁
- 제재 지속 턴, 동맹 유지, 전쟁 소모전 처리
- 즉시 붕괴 조건 + 최종 엔딩 판정
- 저장/불러오기
- 최근 턴 리포트
- 뉴스 피드, 연대기 로그, 세력 관계도, 상태 플래그

## 실행 방법
- 로컬 확인: 브라우저에서 `index.html`을 직접 열면 됩니다.
- 배포 빌드: `npm run build`
- 로컬 검증: `npm test`

## CI/CD
- GitHub Actions + GitHub Pages 기준 자동 배포 워크플로를 포함합니다.
- `main` 브랜치 push 시 테스트 후 `dist/` 산출물을 Pages에 배포하도록 설계했습니다.

## 로컬 LLM 연결
이 프로젝트는 정적 프론트와 별도 로컬 브리지 서버로 분리되어 있습니다.

### 브리지 실행
```bash
cd server
npm install
npm start
```

기본 브리지 주소:
- `http://127.0.0.1:8787`

기본 모델 엔드포인트:
- `http://127.0.0.1:11434/api/generate`

즉, Ollama 같은 로컬 모델 서버를 먼저 띄우고, 그 위에 브리지 서버를 실행하면 턴 종료 후 LLM 브리핑이 자동으로 붙습니다.

## 설계 의도
이번 버전은 단순 데모를 넘어서, 다음 감각을 검증하는 게임다운 MVP를 목표로 했습니다.
- 턴마다 무엇을 포기하고 무엇을 키울지 결정하는 전략성
- 세계 이벤트와 AI 세력 반응 때문에 계획이 흔들리는 비결정성
